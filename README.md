# Java-console-instant-messaging
**`Java`控制台即时通讯系统**

客户端代码：`ChatClient`

服务器代码：`ChatServer`



**已实现功能：**

- [x] 登录、验证
- [x] 私聊消息
- [x] 群发消息
- [x] 私发文件
- [x] 服务器公告
- [x] 留言板
